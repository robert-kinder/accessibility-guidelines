1. Contrast and colour

Contrast

I struggle to read text when it doesn't have good contrast.
  • If text is big (24+ pixels or 18+ pixels and bold) please make sure there's at least 3:1 contrast.
  • If text is not big, please give it a24+ pixels (or 18+ pixels and bold), please give it at least 4.5:1 contrast.
I struggle to see interactive elements when they don't have a good contrast – or know what state they're in.
  • Please make sure that interactive element have a 3:1 contrast.
  • When an interactive element is in a special state (e.g. focused, or in error) please make sure that whatever you use to indicate that status has 3:1 contrast.

Use of colour

I don't differentiate colours but I still need to understand what the page is saying and how to use it.
  • Don't communicate information using just colour.
    • Don't use colour alone to identify data on infographics.
    • If there are links in sentences or paragraphs, show me where they are using more than just colour.
    • If I've made an error in a form, don't use colour alone (e.g. a red border) to show me where the error is.

2. Dynamic size and layout

Page structure

I use a screen reader. I need to understand what's on a page and use all the functionalities.
  • Please use well-nested headings (even if some of them are visually hidden) to organise all the content on the page.
  • Please use tables, when the data can be organised as a table.

3. Structure and navigation

Text resizing

I need to increase the size of text to 200% so I can read it.
  • Please do not prevent me from using pinch and zoom on touch-screen devices.
  • Relative units?
I need to increase the space between lines, paragraphs, words and letters to help me read.

Some explanations.

  • Please do not give limit both the width and height of elements.
  • Please do not prevent me from overriding text styles.

Responsive layout

I need to make the page way bigger, and still have access to everything. If I need to scroll both horizontally and vertically, that makes finding and noticing things on the page almost impossible.
  • Please make sure content will reflow to a single column when zoomed and not produce scrolling in both directions.

6. Language and instructions

Instructions in forms

I need to understand (and remember) what information I'm supposed to put in a text field.
  • Make sure that all form inputs have a label that's associated in code.
  • Make sure that the visible labels don't disappear when information is entered.

Input errors in forms

I need to notice if I've made a mistake in a form.
  • Clearly describe each input error in text. (A red border is not enough).
  • Associate each error with the input that it relates to.

Inclusive instructions

I can't see the page's layout or colours, so instructions that use words like "the button above" or "the green tick" mean nothing to me.
  • Make sure that instructions don't rely on sensory characteristics of the interface.

Language in code

I need my screen reader to pronounce words correctly, otherwise I can't understand what it's saying.
  • Specify the language of the page using the lang attribute on the html element.
  • If the page includes words in a different language, specify that language using the lang attribute on the container element.
